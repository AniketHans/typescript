## Commands

1. `npm init`
2. `tsc --init`
3. `npm i express`
4. `npm i @types/express @types/node`, for intalling the types of a package if available
5. `npm i typescript nodemon --save-dev`, saved as dev packages as they are only used for dev purpose
6. `npm i concurrently --save-dev`, to run tsc -w and nodemon command concurrently
